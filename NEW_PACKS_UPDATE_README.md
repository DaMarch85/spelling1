# Spell Battle Cards: new packs + unlock rules

Upload the included files to GitHub, preserving folder paths.

Then run:

`supabase_update_new_packs.sql`

in Supabase.

## New unlock rules

- Dinosaurs & Prehistoric Beasts stays unlocked by default.
- Users get 1 extra pack unlock free.
- After that, each 7 unique owned cards earns another pack unlock.
- Example: with the free pack, a new user can have 2 packs unlocked immediately. At 7 cards owned, they can unlock another pack. At 14 cards owned, they can unlock another, and so on.
